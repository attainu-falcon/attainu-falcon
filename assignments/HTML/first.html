Different HTML tags

HTML is the language which a browser understands. HTML uses different tags for different purposes.An opening HTML tag is closed by a '/' followed by the tag type. Some tags are self-closing that is it closes in the opening tag itself.

Let's see some tags:

  • Heading tag- from <h1> to <h6>
  • Paragraph tag- <p>
  • Anchor tag-<a>
  • Image tag-<img>
  • Container tags-<div> and <span>

    • There are lots of more tags which we may find while developing websites.
