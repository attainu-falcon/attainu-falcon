Block and Inline elements

HTML tags are divided into two parts as per their apprearamce in the user interface.

  1. Block elements
  2. Inline elements
Let's see these in the table below

Table-1

Different block and Inline elements

Block Elements Inline Elements
<p> <textarea>
<hr> <br>
<div> <span>
<ol> <img>
<ul> <input>
<pre> <button>
Block elements takes up the entire width of the screen in the UI while inline elements takes up the space of the width of the element only.
